# Инструкция по использованию продукта

## Требования

- Python 3.12+
- pip
- (опционально) Docker и Docker Compose

## Установка и запуск

```bash
pip install -r requirements.txt
python tasks/manage.py migrate
python tasks/manage.py runserver
```

После запуска откройте `http://127.0.0.1:8000/`.

## Сценарий первого входа

1. Перейдите на страницу регистрации администратора.
2. Создайте первого администратора (доступно только один раз).
3. Войдите в систему под администратором.
4. Через раздел создания сотрудников добавьте пользователей команды.
5. Создайте проекты и задачи, назначьте исполнителей.

## Роли пользователей

- Администратор:
  - управляет сотрудниками;
  - создаёт, редактирует и удаляет проекты;
  - создаёт задачи и контролирует нагрузку.
- Сотрудник:
  - работает со своими задачами;
  - комментирует задачи и загружает файлы;
  - отслеживает прогресс и статус.

## Автоматическое тестирование

```bash
python tasks/manage.py makemigrations --check --dry-run
python tasks/manage.py test new users
```

## Генерация документации кода

```bash
doxygen Doxyfile
```

Результат: `docs/doxygen/html/index.html`.

## Запуск в Docker

```bash
docker compose up --build
```

Если используется внешний путь к SQLite, задайте переменную `DJANGO_DB_PATH`.
