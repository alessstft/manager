Описание задачи

Необходимо разработать API endpoints для регистрации новых пользователей, входа в систему и обновления профиля. API должно поддерживать JWT-аутентификацию и валидацию входных данных.

Требования:

  • Регистрация с email и паролем
  • Вход с email/паролем
  • Обновление токена
  • Валидация входных данных
  • Документирование с помощью Swagger
